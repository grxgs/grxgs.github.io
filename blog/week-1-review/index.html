Week 1 Review

If you’re following the timeline I set in my last post, you’ll know that week 1 is finally complete. Week 1 went surprisingly well. In the first four chapters, we covered setting up python, variables, lists, tuples, strings, basic numerical operations, and looping.

Getting Python set up on my machine was no problem. Instead of doing everything on my machine locally, I opted to use a virtual machine. I’m using the Ubuntu 14.04 LTS. I’ll be upgrading to 16.04 LTS sometime next week. Having my entire environment on a virtual machine is a lot easier for me.

I feel more comfortable testing and making changes on a system that’s entirely separate from what I use everyday. I used to work with the command line quite a lot so everything is very familiar.

The flip side to this is that I needed to completely configure a command line environment. Fleshing out my .bashrc file has been a fun exercise as I remember all of the CLI shortcuts and CLI programs that are available.

In regards to actual Python, the chapters have been fairly easy. The difficult part of this is reacquainting myself with Python’s syntax. While it is readable the fact that I haven’t really used it or read it in several years is obvious.

The easy part though has been remembering the concepts of everything. Variables are just a way of storing data in a way that makes them easier to access and use later. Lists and tuples are just containers for different types of data. Looping is just a construct that allows us to perform iteration really easily.

The fact that the concepts are coming back so easily is really driving me to read and get through all of the exercises in each chapter. So much so that I’ve already read Chapter 5 well before I’m supposed to. Hopefully, week 2 goes just as well or better than week 1.

Until next time…
